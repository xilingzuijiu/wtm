概述

“微淘米”是一款赚钱的APP，用户通过微淘米去下载应用即可获得现金奖励。微淘米可以对ios（苹果）用户和安卓用户开放。 所有用户都可以通过邀请好友加入微淘米，获得丰厚的邀请与提成奖励！钱包中的现金可轻松提现，小伙伴们反馈都赚High了。

现金奖励

在微淘米APP中通过以下几种方式，可获得现金奖励；

  1. 1.阅读赚钱，进入任务列表，选择文章完成阅读任务，赚取奖励：
  2. 2.关注赚钱，进入任务列表，选择公众号完成关注任务，赚取奖励（关注后相应奖励会到用户的钱包内，但这部分奖励不能到可提现余额中，需连续关注7天后才可提现）；
  3. 3.任务奖励，进入每日任务，完成其中的任务可领取相应奖励；
  4. 4.邀请好友成功，获得好友提成，您的好友完成任务，您可以获得该任务奖励的10%！
关于提现
  1. * 微信每笔提现均会扣取提现金额的0.5%(最低1元最高25元)手续费
  2. * 到账时间最快当天，通常为第二天到账
  3. * 周末、法定节假日不处理提现（星期5-6顺延星期1，星期天顺延星期二）
  4. * 24小时内最多可提现三次，每次提现需间隔10分钟。
  5. * 所有提现均人工审核，审核完成后由系统自动打款
  6. * 邀请的好友注册成功立即发放邀请奖励50米币，好友成功阅读25篇文章后邀请奖励方可提现
无法获得奖励
  1. 1.每天完成任务，任务随时都会变化，所有任务都是限时限量的，建议经常查看
  2. 2.你邀请的好友，完成任务，你都将获得好友收入的10%永久提成奖励
用户邀请码

每个用户都有一个邀请码，在【个人中心 或 邀请好友】查看；邀请的好友在注册时填写邀请码您才能获取提成

账户余额不对?

点击首页我的钱包，可时时刷新您的余额,完成关注任务后相应奖励会到用户的钱包内，但这部分奖励不能立即到可提现余额中，需连续关注7天后才可提现

关于作弊
  1. 1.使用共享ID，多设备、多账号所有非正常方式都视为作弊
  2. 2.无论以什么方式刻意刷任务、刷好友等情况都将视为作弊行为
  3. 3.所有提现均是人工审核，非常严格。任何非正常方式都将导致提现失败，严重的冻结IP、设备与相关联的ID。请用户切勿因小失大
